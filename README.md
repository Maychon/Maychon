🔄 Pipeline ETL — Gestão de Contratos
Problema: Dados críticos de contratos fragmentados em silos (Salesforce, SAP, BigQuery), sem visibilidade unificada para as gerações tomar decisões.
Solução Tech:
Ingestão multifonte via Apache Airflow
Transformação e modelagem com Python + Spark
Destino centralizado no BigQuery (Esquema Estrela)
Camada semântica para consumo analítico
Impacto: Redução de lead time para insights gerenciais de dias para minutos, com fonte dinâmica da verdade para dados contratuais.
`Python` `Fluxo de ar` `Faísca` `GCP` `BigQuery` `SQL`

📊 Papel de Contratos em Tempo Real
Problema: Equipes de gestão sem visibilidade em tempo real do status, vencimentos e riscos do portfólio de contratos.
Solução Tech:
Conexão direta com BigQuery (fonte de verdade)
Modelagem dimensional e KPIs definidos com stakeholders
Materiais interativos em Power BI / Tableau
Alertas automáticos para contratos críticos
Impacto: Monitoramento proativo do porto, redução de contratos vencidos não renovados e aumento da visibilidade executiva.
`Power BI` `Tableau` `BigQuery` `SQL` `DAX`

Problema: Projetos de bonificação realizados manualmente em planos, sem precisão e com alto esforço operacional.

Feature Engineering sobre dados históricos de desenvolvimento
